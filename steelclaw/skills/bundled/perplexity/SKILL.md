# Perplexity

Search the web using Perplexity AI's Sonar model for grounded answers with citations.

## Metadata
- version: 1.0.0
- author: SteelClaw
- triggers: perplexity, search, research, sonar, web search ai

## System Prompt
You can use Perplexity. Credentials must be configured via `steelclaw skills configure perplexity`.

## Tools

### search
Search the web using Perplexity AI.

**Parameters:**
- `query` (string, required): Search query
- `model` (string): Model to use (default: "sonar")
